Инструкции к выполнению летней практики

Суть: решить и защитить свое решение одной из задач сайтa cryptohack.org
  • Необходимо выбрать одну из задач, представленных на сайте cryptohack.org сложностью не менее 50 баллов Список предлагаемых заданий
  • Отметить свою фамилию напротив выбранного задания в google spreadsheet (или добавить любую понравившуюся задачу) и решить её.
  • Работу можно выполнять индивидуально, либо в группе, состоящей из двух человек.
  • Задачи не должны повторяться! Следите за выбором сокурсников. Продублированные задачи приниматься не будут.
  • В субботу 03.07 будет проходить защита работы. Для защиты необходимо подготовить 10-минутное выступление с использованием презентации, оформленной в пакете Beamer. Презентация должна включать как минимум
    1. Титульный слайд с названием задачи и состав группы (макс. 2 человека)
    2. Формулировка задачи
    3. Методы решения задачи
    4. Он-лайн демострация корректности решения
  • Защита будет проводится очно. Время и место будут объявлены позже.
Формирование оценки за практику:

Сложность выбранной задачи + Эффективность решения (макс. 20 баллов) + Презентации (макс. 40 баллов) - Помощь (10 баллов)

В 5ти-бальной:
5 = >100 баллов
4 = 85--100 баллов
3 = 60--85 баллов
Если работа выполнена в группе, участники группы получают одинаковую оценку.
Студент может запросить помощь в решении задачи. Это учитывается при формировании оценки.
