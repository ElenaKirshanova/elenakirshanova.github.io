
1. В таблицах на странице курса найдите свою фамилию и алгоритм, стоящий напротив вашей фамилии.
Ваша задача: запрограммировать этот алгоритм в системе Sage.

2. Для проверки корректности вашей программы следует использовать соответствующую встроенную функцию в Sage.

3. Загрузить готовую программу на созданный вами репозиторий.

4. Написать отчет в LaTeX по сделанному алгоритму (см. Инструкцию №1 для 1-2 курсов для полезных ссылок по LaTeX'у).
Для оформления следует использовать шаблон lec-template.txt со страницы курса.
Отчет должен включать в себя: 
	- ваше ФИО, название алгоритма
	- описание алгоритма (с использованием окружения algorithmic как показано в шаблоне)
	- описание машины, на которой проводились тесты лабораторной
	- сравнение скорости работы вашего алгоритма со встроенным. Для этого сравнения следует сделать несколько тестов на больших входных данных (понятие "больших" зависит от алгоритма) и оформить результаты сравнения в таблицу.

5. Выложить .tex файл отчета на GIT.
